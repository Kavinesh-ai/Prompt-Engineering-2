# Chain-of-Thought (CoT) Logic Engine

**Project 2 – Prompt Engineering Internship @ DecodeLabs**

## Overview
This project demonstrates **Chain-of-Thought (CoT) Prompting**, where AI is guided to solve problems through step-by-step reasoning and a self-correction phase before providing the final answer. The goal is to improve logical reasoning and reduce hallucinations in AI responses.

## Features
- Step-by-step reasoning
- Self-correction mechanism
- Logic and math problem solving
- Hallucination reduction
- Tested on 20+ reasoning challenges

## Skills
- Prompt Engineering
- Chain-of-Thought Prompting
- Logical Reasoning
- AI Evaluation
- Critical Thinking
